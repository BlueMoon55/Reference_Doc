無線LAN簡単接続機能
無線LANアクセスポイント(親機)とパソコンやゲーム機など(子機)との無線LAN接続設定を電話機やパソコン、「らくらくスタートボタン」から簡単に行うことができる機能です。
　
設定方法
設定方法には、以下の3通りの方法があります。
■「Web設定」で設定する
パソコンにインストールした無線LAN端末ユーティリティを使用して接続する場合の例です。
1. 本商品にパソコンを接続し、起動する
2. Webブラウザを起動し、「Web設定」のページを開く
Webブラウザのアドレスに「http://ntt.setup/」もしくは本商品のIPアドレス（「http://192.168.1.1/」　※工場出荷時の場合）と入力後、パソコンの「Enter」キーを押します。
3. ユーザー名に 'user' と入力し、機器設定用パスワードを入力する
ユーザー名は、すべて半角小文字で入力してください。
4. 「Web設定」の「無線LAN設定」-「無線LAN簡単セットアップ」の［おまかせモード］をクリックする
5. 無線LAN接続を行うパソコンにインストールした無線LAN端末のユーティリティから無線LAN簡単接続機能（WPS）を起動する
起動方法は、無線LAN端末の取扱説明書などをご覧ください。
「無線LAN端末の設定が完了しました。」と表示されたら、設定が完了です。
※［プッシュボタン方式］/［PIN方式］をご利用になる場合は、「無線LAN設定」-「無線LAN簡単セットアップ」をご覧ください。
■電話機で設定する
ゲーム機と接続する場合の例です。
1. ゲーム機の「無線LAN簡単接続機能(WPS)」を起動する
※設定方法などの詳細については、ゲーム機の取扱説明書などをご覧ください。
2. 電話機のハンドセットを取りあげる
「ツー」という音を確認します。
3. 電話機のダイヤルボタンを「 」と押す
本商品の登録ランプが緑点滅します。
4. 本商品前面の登録ランプが橙点灯することを確認する
設定が完了すると「設定が完了しました。」と音声ガイダンスが流れ、本商品前面の登録ランプが橙点灯後、緑点灯に変わります。
5. 電話機のハンドセットを置く
■「らくらくスタートボタン」で設定する
パソコンにインストールした無線LAN端末ユーティリティを使用して接続する場合の例です。
1. パソコンにインストールした無線LAN端末のユーティリティから無線LAN簡単接続機能（WPS）を起動する
2. 本商品背面の「らくらくスタートボタン」を1秒以上押し、本商品前面の登録ランプが緑点滅したら離す
3. 本商品前面の登録ランプが橙点灯することを確認する
「無線LAN簡単接続機能」での無線LAN設定が完了し、登録ランプは橙点灯後、緑点灯に変わります。
